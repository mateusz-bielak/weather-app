# Get weather node app

This app allows you to check weather in different locations.

## Prerequisites

To run app on your local machine you will need:

-   [node](https://nodejs.org/en/)

## Installation

    $ npm install

Usage:

    node app.js [option]
    npm start app.js -- [option]

      Options:
        --address, [-a]   Get weather
        --help            Shows help
